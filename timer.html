This is a timer that will count down from a given time to zero. The timer can be paused and restarted, and even have the time changed when paused.

When the timer is stopped use the arrow keys to select the time. Left and right move between the different fields (hours, minutes, and seconds), and up and down move the time in the selected field up or down.

Once the time has been set the timer can be started by clicking on the start button, by left clicking and choosing "start" from the menu, or by use of one of the keys given below. Similarly, it can be paused the same way: the Pause button, the Pause menu entry, or any of the shortcut keys listed below.

Keys

  • Left arrow: when paused, move the input the the field to the left
  • right arrow: when paused, move the input the the field to the right
  • down arrow: when paused, decrease the value in the selected field by 1 (for minutes or second roll over to 59)
  • up arrow: when paused, increase the value in the selected field by 1 (for minutes or second roll over from 59 to 0)
  • Space, Enter: toggle the timer state between running and paused
  • P: pause the timer
  • S, G: start the timer

Config options

  • Foreground: the foreground color
  • Background: the background color
  • Font: Select the font to use for display. (monospaced fonts work best)
  • Bold: Use the bold version of the font, if available
  • Alarm sound: the bell to ring when the alarm goes off. The 2 predefined bells ('ding' and 'bzzzt') are available, as well as 2 special choices, "(none)" and "(default)". "(none)" will not ring a bell when the alarm goes off, and "(default)" will use the value configured for Alarm.ding.
